# HUTECH Timetable App (timetable_app)

Ứng dụng Thời khóa biểu – Lịch thi – Hỏi đáp dành cho sinh viên HUTECH,
phát triển bằng Flutter và sử dụng Firebase.

## Công nghệ sử dụng
- Flutter
- Firebase Authentication (Đăng nhập / Đăng ký)
- Cloud Firestore (Lưu trữ dữ liệu người dùng)

## Tính năng
- Đăng nhập / Đăng ký tài khoản sinh viên
- Trang Home: truy cập nhanh (TKB / Lịch thi / Hỏi đáp)
- Thời khóa biểu
- Lịch thi
- Hỏi đáp – FAQ
- Hồ sơ sinh viên:
    - Cập nhật họ tên
    - Đổi avatar
    - Đổi mật khẩu
    - Cập nhật email
